what is Html ?

HTML, or HyperText Markup Language, is a markup language that defines the structure and content of a web page.

what is DOCTYPE in Html?

A DOCTYPE, or document type declaration, is a line of code that tells a web browser what version of HTML a document is written in.

What is tag? how many types of tags?

The html tag represents the root of an HTML document.There are two kinds of HTML tags: paired and unpaired.

what is element in Html?

An HTML element is a section of an HTML document that represents a part of a web page's content

what is difference b/w the tag & element?

Tags are used to define the beginning and end of an element, while elements consist of the opening tag, content, and closing tag.

what is attribute?

An HTML attribute is a piece of markup language used to adjust the behavior or display of an HTML element.

href: Is a reference to information for the current page ie css info for the page style or link to another page. Page Parsing is not stopped. src: Is a resource to be added/loaded to the page as in images or javascript.

what is the difference b/w block and inline elements?
Block Inline
These elements start on a new line and take up the full width of the page. They appear stacked on top of each other, with each new element starting on a new line. Examples of block elements include div, nav, header, section, and footer. These elements appear in line with the elements they are placed next to and do not start on a new line. They only take up as much width as their content requires. Examples of inline elements include span, input, label, select, and a tags..
what is semantic vs non semantic elements?

In HTML (Hypertext Markup Language), semantic elements clearly indicate the meaning of their content, while non-semantic elements don't. Semantic:These elements provide context and improve the user experience. They help search engines understand the content, and make it easier for developers to maintain. Examples of semantic elements include form, table, and article.
non-semantic:These elements don't provide any information about their content, and are used for styling purposes. Examples of non-semantic elements include div ,etc.

diff b/w div and span ?
div span
Used to group and style larger blocks of code, such as paragraphs or sections, on a web page. Div tags are often used for layout purposes, such as structuring a page with CSS Flexbox or Grid systems, or manipulating sections with JavaScript. Used to group and style inline elements, such as short strings or individual words, without creating a new line. Span tags are often used to apply styles or target a section of text or inline elements with JavaScript.
Table vs li (research)

HTML tables should be used when you have tabular data that needs to be arranged in rows and columns, with a clear relationship between the cells. On the other hand, HTML lists should be used when you have sequential or hierarchical data that needs to be ordered or unordered, with a clear relationship between the items.

what are meta tags?

eta tags are HTML tags that provide information about a page's content to search engines and other clients. They are usually added to the head section of an HTML page and are only visible in the page's source code.

what is use of forms in html?

HTML forms are used to collect user input and send it to a server for processing

what are form attribute

HTML form attributes define the behavior and properties of a form element.

what are iframes?

An iframe, or inline frame, is an HTML element that allows you to embed another HTML document within your own page.

what is the use of an iframe?

An inline frame (iframe) is a HTML element that loads another HTML page within the document. It essentially puts another webpage within the parent page. They are commonly used for advertisements, embedded videos, web analytics and interactive content.

what are the advantages of collapsing in white space?

While you are writing the HTML for your web page you want the code to be more understandable/readable to the users.
Collapsing white spaces decreases the transmission time between the server and the client because collapsing features remove unnecessary bytes that are occupied by the white spaces.
By mistake, if you leave extra white space, the browser will ignore it and display the UI perfectly.

what are the html entities?

HTML entities are character sequences that represent other characters in HTML.

Are the elements & tags are same thing in html?

HTML tags and elements are not the same thing. A HTML tag is like a container for either content or other HTML tags. A HTML element consists of the start tag + content tag + end tag. Some HTML tags do not have end tags like img hence in those cases HTML element will consist of start tag + content
